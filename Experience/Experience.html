Games with Unity and C#

Beat Blocker

Beat Blocker was the final project for my Game Design course at school. Beat Blocker is a rhythm game where the game takes a midi file and gets the timings of the notes and creates objects to represent those notes and sends them down the screen based on the timing of the note.

Bot Beatdown

Bot Beatdown is the second game developed with other students in my class for a class of second graders at a local elementary school. This game is a simple beat 'em up game where you control one of 4 super heroes to defeat a swarm of evil robots. This game was build so that it would function on both PC and mobile with a toggle button in the options to turn on and off the mobile UI.

Super Power Scuffle

Super Power Scuffle is a game that was made at school with a group of other students for a class of second graders at a nearby elementary school. I did all of the programming and assets were supplied by other students in my group. In this game you get to customize a hero and use the hero you made in an infinite runner game.

Tempo Tantrum

Tempo Tantrum is a game that is a personal project of mine. Tempo Tantrum is a fighting game where you can do attacks to a song, the game reads a midi file and sends objects down the screen that the player has to do inputs when they reach a certain point on the screen to attack.
